# ti-markov-russian

Лабораторная по теории информации: подсчёт статистики символов русского языка и цепей Маркова (порядок 1–13) по корпусу текстов в папке `corpus/`.
# Статистические особенности русского языка (цепи Маркова)

## Функционал
- Чтение корпуса из папки `corpus/*.txt`
- Нормализация: 33 русские буквы + .,!? + пробел (нижний регистр)
- Подсчёт частот символов и условных вероятностей для порядков 1..13
- Сохранение в `data/frequencies.sqlite3`

## Таблицы БД
- `freq_overall`: общие частоты всех символов
- `freq_conditional`: P(symbol | context) для каждого порядка
